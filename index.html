libharmony

Overview

libharmony is a C++ library for symbolic domain modeling of music-theory concepts.

The library provides types and operations for representing musical structure, including notes, pitch classes, intervals, scales, modes, and finite sets of pitch classes. Its scope is limited to representation and transformation; it does not perform audio processing, MIDI I/O, or visualization.

libharmony is intended to serve as a small, reusable domain layer that can be embedded in larger systems.

Goals

  • Provide a symbolic model of core music-theory concepts
  • Enable explicit reasoning about scales and pitch-class sets
  • Support transformational and set-based operations on musical structures
  • Remain independent of audio frameworks and application architecture

Building

The project uses CMake and requires a C++20-compliant compiler.

Example build steps:

mkdir .bld
cd .bld
cmake ..
make -j

To list available build targets:

make help

Running

To run the test harness:

./harmony-test
harmony
Definition chord.hpp:17

To run the example executable:

./harmony
